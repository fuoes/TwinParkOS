# TwinParkOS 功能覆盖矩阵

| 业务域 | 功能 | 状态 |
| --- | --- | --- |
| 登录鉴权 | JWT 登录、会话恢复、退出登录 | 可用 |
| 权限管理 | RBAC 菜单权限、接口权限、角色配置 | 可用 |
| 驾驶舱 | 园区 KPI、空间总览、告警、工单、趋势 | 可用 |
| 数字孪生 | 三维场景、图层、楼栋详情、时间轴 | 可用 |
| 虚拟仿真 | 遥测推送、设备控制、故障场景、恢复 | 可用 |
| 设备运维 | 设备、巡检、维保增删改查 | 可用 |
| 能耗管理 | 总览、趋势、排行、企业账单、导出 | 可用 |
| 安防管控 | 视频墙、摄像头、访客、车辆、门禁 | 可用 |
| 环境监测 | 热力图、点位、阈值规则、环境告警 | 可用 |
| 物业工单 | 创建、派单、接单、处理、验收、关闭 | 可用 |
| 企业服务 | 企业档案、服务申请、公告 | 可用 |
| 空间资产 | 楼栋、房间、租赁状态、合同 | 可用 |
| 告警中心 | 模拟、确认、关闭、生成工单、规则 | 可用 |
| 报表中心 | 运营、设备、能耗、告警、工单 CSV | 可用 |
| 系统管理 | 用户、角色、字典、接口、日志、重置 | 可用 |

所有“可用”功能均为本地虚拟演示模式，不依赖真实设备或第三方平台。
